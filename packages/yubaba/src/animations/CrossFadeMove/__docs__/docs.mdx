# CrossFadeMove

Is a composite animation of [CrossFade](/cross-fade) and [Move](/move).
Useful for animating an element moving from one position to another when the elements have **stark visual differences** to each other, but is still of a similar aspect ratio.

## Usage

```js
import { CrossFadeMove } from 'yubaba';
```

<Playground>
  <Common.Toggler>
    {({ shown, toggle }) =>
      shown ? (
        <Baba name="cross-fade-move" key="1">
          <CrossFadeMove>
            {({ ref, style }) => (
              <Styled.Box onClick={() => toggle()} style={style} innerRef={ref}>
                Hello, world!
              </Styled.Box>
            )}
          </CrossFadeMove>
        </Baba>
      ) : (
        <Baba name="cross-fade-move" key="2">
          <CrossFadeMove>
            {({ ref, style }) => (
              <Styled.Box style={style} innerRef={ref} onClick={() => toggle()} alignRight />
            )}
          </CrossFadeMove>
        </Baba>
      )
    }
  </Common.Toggler>
</Playground>
