# Move

Useful for animating an element moving from one position to another,
works best when the element has **very little visual difference**.
This animation uses the FLIP technique so it is very performant.

## Usage

<Playground>
  <Common.Toggler>
    {({ shown, toggle }) =>
      shown ? (
        <Baba name="move" key="1">
          <Move>
            {({ ref, style }) => (
              <Styled.Box onClick={() => toggle()} style={style} innerRef={ref} />
            )}
          </Move>
        </Baba>
      ) : (
        <Baba name="move" key="2">
          <Move>
            {({ ref, style }) => (
              <Styled.Box style={style} innerRef={ref} onClick={() => toggle()} alignRight />
            )}
          </Move>
        </Baba>
      )
    }
  </Common.Toggler>
</Playground>
