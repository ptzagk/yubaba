# Getting started

## Moving to another element

The bread and butter animation in `yubaba` is moving an element.
Simple enough - after a state change we can automatically have the element animate into the new position.
We can do this using the [Baba](/baba) and [Move](/move) components.

For this example we are animating from one element to another over a state change.

<Playground>
  <Styled.Container>
    <Toggler>
      {({ shown, toggle }) =>
        !shown ? (
          <div>
            <Baba name="move">
              <Move>
                {({ ref, style }) => (
                  <Styled.Thumbnail style={style} onClick={() => toggle()} innerRef={ref} />
                )}
              </Move>
            </Baba>
          </div>
        ) : (
          <Baba name="move">
            <Move>
              {({ ref, style }) => (
                <Styled.Banner style={style} onClick={() => toggle()} innerRef={ref} />
              )}
            </Move>
          </Baba>
        )
      }
    </Toggler>
  </Styled.Container>
</Playground>

## Moving to the same element

Sometimes we want to animate to and from the same element over a state change,
to do that we can abuse how `key`'s work in react - we just update the key at the same time as the state change.

<Playground>
  <Styled.Container>
    <Toggler>
      {({ shown, toggle }) => (
        <Baba name="same-move" key={shown}>
          <Move>
            {({ ref, style }) => (
              <Styled.Thumbnail
                style={style}
                onClick={() => toggle()}
                innerRef={ref}
                appearance={shown ? 'big' : 'small'}
              />
            )}
          </Move>
        </Baba>
      )}
    </Toggler>
  </Styled.Container>
</Playground>

## Moving to a distinct other element

Sometimes we can't have the affordances of two elements visually looking the same,
for that we can use the [CrossFadeMove](/cross-fade-move) animation to transition between them.

<Playground>
  <Styled.Container>
    <Toggler>
      {({ shown, toggle }) =>
        !shown ? (
          <div>
            <Baba name="cross-fade-move">
              <CrossFadeMove>
                {({ ref, style }) => (
                  <Styled.Thumbnail style={style} onClick={() => toggle()} innerRef={ref} />
                )}
              </CrossFadeMove>
            </Baba>
          </div>
        ) : (
          <Baba name="cross-fade-move">
            <CrossFadeMove>
              {({ ref, style }) => (
                <Styled.Banner
                  style={style}
                  onClick={() => toggle()}
                  innerRef={ref}
                  background="#ED2939"
                />
              )}
            </CrossFadeMove>
          </Baba>
        )
      }
    </Toggler>
  </Styled.Container>
</Playground>
